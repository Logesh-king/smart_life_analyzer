# Smart Life Analyzer

A comprehensive life tracking and analytics platform built with Django.

## Features

- **Daily Entry Tracking**: Log sleep, work, expenses, mood, and notes
- **Analytics Dashboard**: Visual insights with charts and statistics
- **AI Suggestions**: Personalized recommendations based on your data
- **Goal Setting**: Set and track personal goals
- **Export Data**: Download your data in JSON format
